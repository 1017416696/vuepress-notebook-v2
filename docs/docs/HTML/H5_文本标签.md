# 文本标签

## ruby

使用`<ruby>`标签来标记含有注音或其他额外信息的文字

```html
<ruby>  
  <span>魑魅魍魉</span>  
  <rt>chī méi wǎng liǎng</rt>  
</ruby>  
<ruby>  
  <span>魑</span>  
  <rt>chī</rt>  
</ruby>
```

## mark 

`<mark>`标签用于在页面中突出显示文本，通常用于标注相关或重要的内容 

`<mark>`标签的使用场景用于在==搜索结果==中高亮显示搜索词

`<mark>`标签只用于标识需要高亮的文本，而不应该用于强调或加粗文字

```
<mark>这段文本需要突出显示。</mark>
```
