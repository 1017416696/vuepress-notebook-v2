## 块元素(block)

在页面中独占一行 ，从上到下排列

默认宽度撑满整个父元素 

默认高度由内容撑开 

可以通过 CSS 设置宽高

![[Pasted image 20230329093349.png]]

## 行内元素 (inline-block)

不独占一行 ，从左到右排列 

默认宽度由内容撑开 

默认高度由内容撑开

==无法==通过 CSS 设置宽高 

![[Pasted image 20230329093411.png]]

浏览器见到 br 元素就会另起一行

span 也是行内元素 

>调整行内元素和行内块元素的位置，可以当作调整文本的位置一样 

## 行内块元素

不独占一行 ，从左到右排列 

默认宽度由内容撑开 

默认高度由内容撑开

可以通过 CSS 设置宽高
