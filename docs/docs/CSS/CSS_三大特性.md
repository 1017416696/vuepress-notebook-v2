# 层叠性 

同一个样式名，被设置了不同的值

层叠性为了解决样式冲突问题

# 继承性 

元素会自动拥有其父元素或者祖先元素上设置的某些样式 

优先继承离得近的

常见的可继承属性：text-? font-? line-? color 

[[CSS_文本属性_行高_line-height]]

MDN 是可查询某一属性是否被继承 

在浏览器中也能够看到继承的属性，如果被继承的属性画上了横线，说明该属性被覆盖了
